# EKO 图片反推助手浏览器插件

这个插件只负责从网页选择图片并发送到本地 EKO 图片反推工具。插件不保存 API Key、不保存独立历史记录，分析结果会自动写入桌面软件的历史记录。

## 安装方式

1. 先打开 EKO 本地软件。
2. 在 Chrome 打开 `chrome://extensions/`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择仓库中的 `browser-extension` 文件夹。

## 使用方式

1. 在网页图片上悬停，点击出现的“反推”按钮。
2. 或右键图片，选择“用 EKO 反推这张图片”。
3. 等待分析完成，结果会显示在页面面板中，同时自动写入 EKO 的历史记录。

## 常见问题

- 提示无法连接：确认 EKO 桌面软件正在运行，再重试。
- 图片无法分析：确认图片地址可公开访问，或改用悬停按钮让插件先读取图片。
- 修改插件代码后：回到 `chrome://extensions/`，点击插件卡片上的刷新按钮。

## 连接方式

桌面软件启动后会在本机监听：

`http://127.0.0.1:17621`

插件会调用：

- `GET /health` 检查本地软件是否打开。
- `POST /analyze` 发送图片给本地软件分析。

接口只监听本机地址，插件不会读取或暴露本地软件里的 API Key。
