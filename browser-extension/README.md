# EKO 图片反推助手 Chrome 插件

这个插件用于把网页里的图片发送到本地 EKO 图片反推工具。插件本身不保存 API Key，也不保存独立历史记录；分析结果会自动写入 EKO 桌面软件的历史记录。

## 安装

1. 先安装并打开 EKO 桌面软件，确认设置中心里的 API 可以正常分析图片。
2. 打开 Chrome：`chrome://extensions/`。
3. 开启右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择仓库里的 `browser-extension` 文件夹。

## 使用

- 网页图片上悬停，点击浮出的“EKO 反推”按钮。
- 或者右键图片，选择“用 EKO 反推这张图片”。
- 结果面板支持 GPT Image / Nano Banana 两套 Prompt 和中英文切换。
- 分析失败时可以直接点“重试”。
